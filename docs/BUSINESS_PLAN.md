# 사과농장 플랫폼 — 사업계획서 구조

> **타겟 사업**: 농식품 벤처육성 지원사업 (첨단기술) ②-3 빅데이터·AI·클라우드
> **지원 규모**: 기업당 최대 3억원 (국고 70% + 자부담 30%)
> **작성 기준**: 2026년 상반기 공고 대비

---

## 1. 사업 개요

### 1.1 플랫폼 소개

| 항목 | 내용 |
|------|------|
| **명칭** | 사과농장 (Smart Orchard Platform) |
| **URL** | https://apple-farm.pages.dev/ |
| **목적** | 사과 재배 전 과정을 데이터 기반으로 지원하는 농업인 전문 플랫폼 |
| **타겟 유저** | 사과 농업인 (신규 귀농~기존 농가), 농업 법인, 지자체 |
| **핵심 기능** | 밭 자동설계 + 수익 시뮬레이션 (국내 유일) |

### 1.2 킬러 피처

**밭 자동설계 + 수익 시뮬레이션**: 면적·품종·지역을 입력하면 최적 배치를 자동 계산하고, 3가지 시나리오(낙관/기본/비관)로 5년 수익을 시뮬레이션합니다. 이 두 기능을 결합한 서비스는 국내에 없습니다.

### 1.3 핵심 가치

- **의사결정 지원**: "어디에, 무슨 품종을, 몇 그루 심을까?" → 데이터 기반 답변
- **리스크 관리**: 기후·가격 변동 시나리오별 수익 비교
- **정보 접근성**: 60대 농업인도 쉽게 쓸 수 있는 극도로 단순한 UI

|------|----------|------|

## 3. 시장 분석

### 3.1 시장 규모

| 지표 | 수치 |
|------|------|
| 국내 사과 농가 수 | 약 28,000호 (2024 기준) |
| 사과 재배 면적 | 약 33,000ha |
| 사과 농업인 평균 연령 | 약 63세 |
| 사과 생산액 | 약 2조원/년 |
| 사과 도매시장 거래액 | 약 1.2조원/년 |

### 3.2 시장 트렌드

- **기후변화 → 재배지 북상**: 경북(전통 적지) → 충북·강원 품질 상승, 남부는 열대작물 전환
- **고령화 + 인력난**: 외국인 노동자 의존도 증가, 적과·수확기 인력 확보가 농사 성패 좌우
- **스마트팜 정책 확대**: 정부 스마트농업 투자 매년 증가, 농업 디지털 전환 가속
- **소비자 트렌드**: 이력추적, 프리미엄 브랜딩, 직거래 수요 증가

### 3.3 경쟁 환경

| 경쟁사/서비스 | 제공 기능 | 차별점 |
|-------------|----------|--------|
| 팜모닝 | 시세, 날씨, 뉴스 | 범용 농업 — 사과 특화 아님 |
| 그린랩스 | 시세, 장부 | 도구 중심 — 설계·시뮬레이션 없음 |
| 농촌진흥청 농사로 | 기술 정보 | 정보 제공 — 의사결정 도구 없음 |
| **사과농장 (본 서비스)** | **밭 설계 + 시뮬레이션 + 시세 + 방제** | **국내 유일 사과 특화 통합 플랫폼** |
