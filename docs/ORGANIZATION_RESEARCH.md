## 1. 국립농산물품질관리원 (NAQS)

> 기존 API_RESEARCH_REPORT.md 섹션 8.1 보완 -- 기존 보고서에서는 "(미검증)" 표시였으나 라이브 검색으로 확인됨

### 1.1 기관 개요

| 항목 | 내용 |
|------|------|
| **정식 명칭** | 국립농산물품질관리원 (National Agricultural Products Quality Management Service) |
| **상위 기관** | 농림축산식품부 소속기관 |
| **공식 URL** | https://www.naqs.go.kr |
| **핵심 역할** | 농산물 안전성조사, 친환경농산물인증, GAP인증, 원산지/GMO 표시관리 |

### 1.2 사과 품질 등급 기준 (농산물표준규격)

**법적 근거**: 농산물 품질관리법에 따른 [농산물표준규격](https://www.law.go.kr/%ED%96%89%EC%A0%95%EA%B7%9C%EC%B9%99/%EB%86%8D%EC%82%B0%EB%AC%BC%ED%91%9C%EC%A4%80%EA%B7%9C%EA%B2%A9/)

사과 표준규격은 다음 기준으로 등급을 분류한다:
- **외관**: 색택, 형상, 손상 정도
- **크기**: 직경/중량 기준 L, M, S 등급
- **내부 품질**: 당도(Brix), 산도, 경도
- **결점 허용 범위**: 등급별 상한

**PJ18_APPLE 활용 방안**:
- 품종 DB에 표준규격 등급 기준을 내장
- 수익 시뮬레이션에서 등급별 가격 차이를 반영 (특/상/보통 가격 괴리 20~40%)
- "이 품종을 이 토양에서 재배하면 특품 비율 예상: XX%" 형태의 분석 제공

### 1.3 GAP (농산물우수관리) 인증 시스템

| 항목 | 내용 |
|------|------|
| **GAP 정보서비스** | https://www.gap.go.kr |
| **API 제공** | **확인됨** - NAQS 데이터 API 포털에서 GAP 인증정보 API 제공 |
| **API URL** | `https://www.naqs.go.kr/dataapi/preview/viewData.do?INF_ID=GAPCERTINFO` |
| **데이터 내용** | 인증 농가 정보, 인증 품목, 인증 기간, 인증 기관 등 |

**GAP 인증 데이터 구조** (확인됨):
- 인증번호
- 인증 농가/단체명
- 대표 품목
- 인증 유효기간
- 인증 기관
- 소재 주소

**PJ18_APPLE 활용 전략**:
```
1. GAP 인증 사과 농가 데이터를 수집하여 "인증 농가 맵" 구현
2. 커뮤니티에서 GAP 인증 농가 배지 시스템
3. 마켓플레이스(Phase 4)에서 GAP 인증 상품 프리미엄 표시
4. GAP 인증 절차 가이드 콘텐츠 제공 (신규 농업인 타겟)
```

### 1.4 농산물이력추적관리 (Traceability) 시스템

| 항목 | 내용 |
|------|------|
| **전자민원시스템** | https://www.agrin.go.kr |
| **공공데이터포털 API** | **확인됨** - `농림축산식품부 국립농산물품질관리원_농산물이력추적등록정보 OpenAPI` |
| **data.go.kr ID** | https://www.data.go.kr/data/15101223/openapi.do |

**이력추적등록정보 API 제공 데이터**:
- 등록번호
- 등록기관
- 단계구분 (생산/유통)
- 생산자단체
- 대표자명
- 유효기간
- 대표품목
- 주소
- 등록농가수
- 등록필지
- 재배면적

**PJ18_APPLE 활용 전략**:
```
1. 이력추적 등록 농가 데이터로 사과 농가 분포 지도 구축
2. 이력추적 번호와 플랫폼 생산이력 데이터 연동
3. 소비자 신뢰도 강화를 위한 이력추적 QR코드 통합 (Phase 4 마켓플레이스)
4. 재배면적 데이터로 지역별 생산규모 분석
```

### 1.5 사과 관련 검사/등급 기준

NAQS는 다음 검사 서비스를 수행한다:
- **출하 전 품질 검사**: 출하 전 등급/규격 적합 여부 확인
- **잔류농약 안전성 검사**: PLS(허용물질목록관리제도) 기반
- **원산지 표시 관리**: "영주사과", "청송사과" 등 지리적표시 관리

**지리적표시 데이터 활용**:
- 영주사과, 청송사과, 안동사과 등 지리적표시 등록 현황
- 프리미엄 브랜드 콘텐츠 및 지역별 차별화 정보 제공

### 2.2 과수생육품질관리시스템 (fruit.nihhs.go.kr) -- 핵심 신규 발견

**핵심 기능 및 데이터**:

| 기능 | URL 패턴 | 제공 데이터 | PJ18 활용도 |
|------|----------|------------|------------|
| **재배적지 지도** | `/apple/agdcm/cultiLandMap.do` | 사과 재배 적합 지역 지도 (기후변화 반영) | **최상** - 밭 설계 시 적지 판단 |
| **만개기 정보** | `/apple/grwhInfo_eclnBlpr.do` | 지역별 사과 만개기 관측 데이터 | **상** - 수분/서리 경보 캘린더 |
| **과실품질정보** | `/main/qlityInfo_frutQlity.do` | 품종별 당도, 산도, 경도 등 품질 데이터 | **상** - 품종 DB 보강 |

| **농작업일정** | `/apple/openApi/farmWorkingPlanNew.do` | 월별 사과 작업 일정 | **최상** - 캘린더 데이터 소스 |
| **농업기술정보** | `/apple/openApi/cropTechInfo.do` | 사과 재배 기술 가이드 | **상** - 기술 콘텐츠 |
| **적산온도 정보** | - | 생육 예측용 적산온도 | **상** - 병해충 발생 시기 예측 |

**특별 참고**: 이 시스템은 Open API 형태로 `농작업일정`과 `농업기술정보`를 제공한다 (URL에 openApi 포함). data.go.kr이 아닌 자체 API 체계일 수 있으므로 별도 확인 필요.

**PJ18_APPLE 통합 전략**:
```
즉시 활용 가능:
  1. 재배적지 지도 데이터 → 밭 설계 "적지 점수" 산출의 공식 근거
  2. 농작업일정 API → 월별 캘린더의 공식 데이터 소스
  3. 과실품질정보 → 품종 DB의 품질 특성 데이터

중기 활용:
  4. 만개기 데이터 → 서리 경보/수분 타이밍 알고리즘 보정
  5. 적산온도 → 해충 발생 예측 모델의 입력값
  6. 재배변동정보 → "기후변화와 사과 재배" 콘텐츠 시리즈
```

### 2.4 사과 품종 개발 및 기후변화 대응

> 라이브 검색으로 확인된 최신 정보

#### 기후변화에 따른 재배지 북상 현황

**핵심 사실** (2025년 기준):
- 사과 재배 적지가 **지속적으로 북상** 중
- **2070년대 전망**: 강원도 일부 지역에서만 재배 가능할 것으로 예측
- 강원도 철원, 양구, 화천까지 10ha 이상 사과 재배지 확대
- 대구/경북 전통 산지에서 고온으로 인한 착색 불량, 일소(햇볕 데임) 피해 증가

출처: [농촌진흥청 - 온난화로 미래 과일 재배 지도 바뀐다](https://www.rda.go.kr/board/board.do?mode=view&prgId=day_farmprmninfoEntry&dataNo=100000778531)

#### 신품종 개발 (지역 맞춤형)

| 품종명 | 특징 | 보급 계획 |
|--------|------|-----------|
| **골든볼** | 노란색 껍질, 착색 관리 불필요, 당도 15Brix | 2025년 5ha, 묘목 15,000그루 보급 |
| **아리수** | 농촌진흥청 개발, 한국 기후 최적화 | 보급 확대 중 |
| **시나노골드** | 일본 품종, 고온 지역 적합 | 경남 등 남부 지역 확대 |

출처: [정책브리핑 - 지역 맞춤형 사과 품종으로 기후변화 준비](https://www.korea.kr/briefing/policyBriefingView.do?newsId=156589892)

**PJ18_APPLE 활용 전략**:
```
1. 품종 DB에 기후변화 대응 신품종 정보 우선 포함

3. 재배적지 변화 시각화 콘텐츠 (시계열 지도 애니메이션)
4. "기후변화 대응 과수원 설계" 프리미엄 기능
```

### 3.3 시장 분석 데이터 활용

**KREI가 제공하는 분석 프레임워크**:

2. **가격 전망**: 전년 동기 비교, 계절성 분석, 수급 기반 예측
3. **정책 영향 분석**: 보조금/지원사업 변동이 생산에 미치는 영향
4. **기후변화 영향**: 재배지 변동, 생산성 변화 장기 전망

**PJ18_APPLE 차별화 기회**:

- 데이터 접근성 낮음 → PJ18_APPLE이 **모바일 알림**으로 핵심 정보 푸시
