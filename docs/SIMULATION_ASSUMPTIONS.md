# 시뮬레이션 주요 가정 (투명화 문서)
> 이 시뮬레이션의 모든 핵심 가정을 명시합니다.
> 실제 농가 상황과 차이가 있을 수 있으며, 투자 판단의 근거로 사용하지 마세요.

---

## 면책 고지
이 시뮬레이션은 **참고용 추정치**이며, 실제 과원 수익은 기후, 병해충, 시세 변동,
개인 재배 기술 등에 따라 크게 달라질 수 있습니다.

---

## 1. 재식 밀도

| 가정 | 값 | 근거 |
|------|---|------|
| 나무 간격 | 5m × 3m | 일반적인 M.9 왜성대목 밀식재배 기준 |
| 유효 면적 비율 | 85% | 도로, 작업로, 집수구 등 제외 |
| 나무 수 계산 | `면적(m²) × 0.85 / 15` | 자동 계산, 사용자 직접 입력 시 우선 |

**한계**:
- 대목 종류(M.9 vs M.26 vs 실생)에 따라 재식 간격이 3×1.5m~6×4m까지 차이
- 지형(경사지)에 따른 유효 면적 변동 미반영

| | 농기계 감가상각 | 200,000원 |
| | 지주·시설 | 100,000원 |
| | 유류·전기료 | 120,000원 |
| | 농작물재해보험 | 80,000원 |
| **합계** | | **2,860,000원/10a** |

**유목기 비용 감소 공식**: `비용 = 성목비용 × (0.5 + 0.5 × min(수확비율 + 0.3, 1.0))`
- 1~2년차: 성목 비용의 65% (수확은 없지만 기본 관리비 필요)
- 3년차: 70%
- 5년차: 90%
- 8년차~: 100%

**한계**:
- 인건비 상승 추세 미반영
- 지역별 임차료 차이 큼 (평지 vs 산간)
- 자가 노동 vs 고용 노동 구분 없음

## 개선 계획
1. 농촌진흥청 과수 경영 분석 자료와 교차 검증

3. 기상 데이터 기반 재해 리스크 시나리오 추가
4. 사용자가 가정값을 직접 수정할 수 있는 UI 제공
